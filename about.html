AI Interest Rate Predictor

A machine learning project that predicts loan interest rates
using an Artificial Neural Network trained on real lending data.

This project uses a Feed-Forward Neural Network (ANN) built with Python and scikit-learn. It takes loan details like principal amount, income, and credit score as input and predicts the annual interest rate (APR) along with EMI and full repayment breakdown.
